# Operação Qualidade de Código – Code Review no GitHub

## Descrição do Projeto
Este projeto tem como objetivo simular um fluxo de trabalho profissional de desenvolvimento utilizando o GitHub. A atividade demonstra como equipes de desenvolvimento utilizam ferramentas como branches, pull requests e code review para garantir a qualidade, segurança e organização do código.

Durante o projeto, um integrante atua como Líder do Projeto, responsável por realizar alterações no código e propor melhorias, enquanto o outro atua como Revisor de Qualidade, responsável por analisar as mudanças, comentar, aprovar ou solicitar correções antes da integração ao código principal.

O objetivo é aplicar boas práticas de colaboração no desenvolvimento de software e compreender a importância do processo de revisão de código em projetos profissionais.
